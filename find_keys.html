Extracting the Key Manually

If no decryption key exists for your game, you can extract it manually using a hex editor. Here is how to find and format the key hidden inside Digital Eclipse game executables.

1 Locate the Key Sequence

  1. Open the game executable file (e.g., MMPR_RR.exe) in HxD or any hex editor.
  2. Search for the following hex pattern identifying the start of the key logic:
    50 C7 87 E8
  3. Locate the sequence in the file. After this sequence, you will find three null bytes (00 00 00), making the full prefix sequence:
    50 C7 87 E8 00 00 00
  4. The decryption key is located in the four bytes immediately following this sequence.
  5. After the key, you will always find the sequence 48 85 DB 74. The key is sandwiched safely in between!
00A14F20 .. .. 50 C7 87 E8 00 00 00 3B 89 5E FA 48 85 DB

2 Change the Endianness

Because of how the architecture stores memory, the hexadecimal bytes are ordered backwards (Little Endian). To use this key in the Pybunga Decryptor, you must reverse the order of the bytes and prefix it with 0x.
